# Simple Keylogger

This project is a basic Python keylogger that records and logs keystrokes to a file. It captures both regular and special keys with timestamps.

## Features

* Records all keystrokes in real time
* Logs both regular keys and special keys (Shift, Enter, Ctrl, etc.)
* Timestamps every keystroke
* Saves logs to a text file

## Technologies Used

* Python

## How to Run

1. Install Python on your system.
2. Install the required library:

```bash
pip install pynput
```

3. Run the program:

```bash
python keylogger.py
```
